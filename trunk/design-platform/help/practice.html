Practice

This tab is used to add practice blocks in the experiment.

Once the blocking strategy chosen an experiment is organized as a hierarchy of blocks. Using this tab, practice blocks can thus be inserted before any block.

Practice blocks can be inserted at the beginning of the experiment (Experiment level) or before each experiments block of a given depth in the hierarchy of blocks (Blocki level). Suppose the factors T and ID are both nested at i level. If T has two different values and ID three, there are six different types of blocks at level i. Inserting practice blocks at the Blocki level means that theses practice blocks will be presented each time a block of a type that has never been presented yet begins. By default, if the blocks are repeated several times, practice blocks are presented only the first time a block occurs but by checking the box 'repetitions' they will be presented before all blocks, including those that are repetitions of blocks already presented.

Inserting a practice block at level i is achieved by letting the checkbox 'off' as unchecked. Then the practice block is defined by:
- the number of trials it contains,
- the values of the different factors for the trials it contains. For each factor, you can choose to show (i) a sample of the values it can take, (ii) a fixed given value (e.g. T=Technique1) or (iii) a fixed value being the value of the factor in the next coming block.
- the kind of block. A practice block can be made of trials similar to the experimental trials or it can be mode of trials of another kind. For example, experimental trials can be 2D pointing tasks while practice trials can be 1D pointing tasks to simply make the participant get used with the acceleration of the pointing device.

Of course, a practice session is not especially limited to a single block. The '+' button at the end of the line allows to add a block at the same level. Also, if there are block repetitions in the experiment, the practice session can appear before each repetition or not by checking or unchecking the 'repetitions' checkbox.
